# 📡 Data Communications
> **Course:** CSE 3227 — Data Communication
> **Chapter:** Introduction to Data Communications
 
---
 
## 📌 What is Data Communication?
 
- Exchanging data between **two or more devices**
- Data is sent through a **transmission medium** (wire cable, fiber optic, or wireless)
- Communication = transferring information from **one place to another**
---
 
## 🎯 Goals of Communication
 
| Goal | Description |
|------|-------------|
| ⚡ Efficient Transfer | Data should be delivered as fast as possible |
| ✅ High Reliability | Data must be **correct and accurate** |
| 🔒 Secure Communication | Data should be protected from unauthorized access |
 
---

## 🖥️ Communication System
 
> A **communication system** is a group of components that work together to **transfer and process information**.
 
### Types of Communication System
 
1. **Analog** Communication System
2. **Digital** Communication System
---

## 🔧 Components of a Communication System
 
### 1. 🎙️ Input Transducer
- Converts the original message into an **electrical signal**
- **Example:** A microphone converts voice → electrical signals
---
 
### 2. 📤 Transmitter
Prepares the signal for transmission. Main functions:
 
| Function | Role |
|----------|------|
| Modulation | Encodes signal for transmission |
| Coding | Represents data in a suitable format |
| Amplification | Boosts signal strength |
| Filtering | Removes unwanted frequencies |
 
---
 
### 3. 📶 Channel
The **path** through which the signal travels.
 
**Examples of channels:**
- Free space (air)
- Coaxial cable
- Fiber optic cable
- Waveguide
> ⚠️ **Signal quality may decrease due to:**
> - **Noise** — unwanted signals
> - **Interference** — external signal disruption
> - **Distortion** — signal shape deformation
 
---
 
### 4. 📥 Receiver
Receives the signal and **recovers the original information**. Main functions:
 
| Function | Role |
|----------|------|
| Amplification | Strengthens the received signal |
| Demodulation | Extracts original data from the signal |
| Filtering | Cleans up the signal |
 
---
 
### 5. 🔊 Output Transducer
Converts the electrical signal into a **usable form**.
 
**Examples:**
- Loudspeaker
- Computer (PC)
- Tape recorder
